blue-sapphire-cs

Package Name:
fifthtry.github.io/blue-sapphire-cs

blue-sapphire-cs is a color scheme which can be used to change color scheme of web packages via using FTD lang.

To use blue-sapphire-cs color scheme on your web package, add below into FPM.ftd file:

-- fpm.dependency: fifthtry.github.io/blue-sapphire-cs as blue-sapphire

-- fpm.dependency: fifthtry.github.io/blue-sapphire-cs as blue-sapphire
Import blue-sapphire into FPM/config.ftd file.
-- import: blue-sapphire

-- import: blue-sapphire
